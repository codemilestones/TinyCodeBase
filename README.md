# TinyCodeRAG

本项目扩展于 TinyRAG：https://github.com/datawhalechina/tiny-universe/tree/main/content/TinyRAG

为了更深入的优化RAG，本项目在TinyRAG的基础上，添加以下功能：

✅ 代码智能分块：专门解析代码数据结构，构建精准向量集  
✅ 开箱即用：提供专用的测试API key  
✅ 模块化测试：每个组件都有独立测试用例，更适合新手同学学习  
✅ 对话体验优化：完整支持多轮上下文对话
